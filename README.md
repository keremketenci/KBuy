## Installation
```
git clone https://github.com/keremketenci/KBuy.git
cd KBuy

composer install
npm install
npm run build

php artisan migrate (create database folder)
php artisan migrate:fresh
php artisan db:seed
```
## Usage
```
composer run dev
```
